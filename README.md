# voicesfromthevoid

sound art installation

# useful commands

List out PLAYBACK Hardware Devices

```
aplay -l
```

Enable the audio output

```
amixer -D hw:CARD=audioinjectorpi set 'Output Mixer HiFi' unmute
```
